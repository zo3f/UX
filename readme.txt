server met 2 frontends, die informatie uit een database halen, toevoegen, bewerken en verwijderen & zoekfunctie in database
Omgezet naar bevolkingsregister met burgers, huwelijken en kinderen toevoegen. Bij huwelijk wordt een inteeltcheck gedaan.

Package.zip weer uitpakken in dezelfde map als server.js
in deze map een folder public maken, waarin burger.html en ambtenaar.html en de bijbehorende scriptbestanden komen te staan.
bevolkingsregister_db moet worden geimporteerd via myphpadmin.
